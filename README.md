# db-decathlon
a simple database for practice SQL

## Exercices SQL

Le plus simple pour travailler chez vous :

Téléchargez DBeaver (gratuit et multiplateforme) :
https://dbeaver.io/

### Pourquoi SQLite ?

- Pas besoin d’installer un serveur SQL (MySQL, MariaDB, etc.)
- Tout fonctionne dans un simple fichier .sqlite
- Très pratique pour travailler hors réseau ou sur un ordinateur personnel
- Idéal pour manipuler des bases de données comme dans les exercices de BTS ou STMG

### Comment utiliser le fichier SQL fourni :

- Lancez DBeaver
- Ouvrir une nouvelle connexion SQLite
- Indiquez votre fichier .sqlite (par exemple exo1.sqlite)
- Les tables et données existent déjà
- Vous pouvez ensuite faire vos requêtes (bouton SQL).

Astuce :
- Chaque exercice est indépendant : créez un fichier .sqlite par exercice si vous voulez rester organisé.
